.. function:: count(self, where)

    Counts the number of rows which meet given criteria.


    :Parameters:

        **where** : function

        ..

            |UDF| which evaluates a row to a boolean




    :Returns:

        : int

        ..

            number of rows for which the where |UDF| evaluated to True.
